'Silverlight Dynamic Data Display' is a library of Silverlight controls for dynamic data visualization.
This is Silverlight version of 'WPF DynamicDataDisplay', a powerfull WPF charting library.

Homepage - http://www.codeplex.com/dynamicdatadisplay/
